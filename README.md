# Used Car Marketplace Backend

A scalable backend application for a used car marketplace built with Java and Spring Boot. The system provides secure JWT-based authentication, car listing management, advanced search and filtering, image uploads to AWS S3, bulk listing uploads through Excel files, and event-driven notifications using Apache Kafka.

## Features

* User Registration & Login
* JWT Authentication & Authorization
* Email Verification & Password Reset
* Car Listing Management
* Dynamic Search & Filtering
* Multi-Image Upload to AWS S3
* Bulk Car Upload via Excel
* Kafka-Based Event Processing
* Dockerized Deployment
* API Documentation using Swagger

## Tech Stack

* Java 21
* Spring Boot
* Spring Security
* Spring Data JPA
* MySQL
* AWS S3, RDS, CloudWatch, SNS
* Apache Kafka
* Docker
* Jenkins
* Swagger
